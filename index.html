Ambienators

Smart Ambient Sensor System for NUS Orbital 2014 project

View project onGitHub

1. Ideation

The SmartAmbient sensor system is an orbital project planned for the Gemini level. It can be used by the managers of workplaces, schools and offices to monitor the usage of utilities such as air conditioning and lighting. It can also serve as a security feature, by monitoring motion in the vicinity. Leaving the lights and air-conditioner wastes electricity as well as money. With this app, the users can get real-time data and alerts based on customised settings (such as when the temperature is below a certain value).

Sensor components will include

  • Temperature Sensor Module
  • Light Sensor Module
  • Motion Sensor Module

2. Planned features

Hardware Design We plan to integrate these components together into a single integrated unit which the user can purchase and install on the ceiling or wall of a room.

  • Arduino Uno Unit
  • Temperature Sensor Module
  • Light Sensor Module
  • Motion Sensor Module
  • Ethernet Module
  • Wifi Module We plan to enable this single integrated unit to connect to the internet, either by using a wired LAN connection, or wifi. The unit will communicate with the Google cloud server, and store ambient statuses onto the cloud, which are accessible to the end-user through a google webapp which we plan to develop. We plan to have at least one ambient sensor working by Milestone 2, and all three ambient sensors working by Milestone 3.

Software Design

We plan to develope a Google WebApp which allows the end-user to: View the instantaneous statuses of the three main sensor modules. Statuses include temperature in degrees Celsius, light intensity in lumens, Boolean presence of motion. (We plan to complete this by Milestone 2)
View the history of the statuses of the three main sensor modules (We plan to complete this by Milestone 3) Allow end-users to preconfigure situations to be alerted.
